## ?? **API Endpoints (Gelecek)**

E?er API Controller eklemek isterseniz:

```csharp
// CoreBuilder/Controllers/TeachersController.cs
[ApiController]
[Route("api/[controller]")]
public class TeachersController : ControllerBase
{
    private readonly TeacherService _teacherService;

    [HttpGet]
    public async Task<ActionResult<List<Teacher>>> GetTeachers(
        [FromQuery] Guid tenantId)
    {
        var teachers = await _teacherService.GetActiveTeachersAsync(tenantId);
        return Ok(teachers);
    }

    [HttpPost]
    public async Task<ActionResult<Teacher>> CreateTeacher(
        [FromBody] Teacher teacher)
    {
        var created = await _teacherService.CreateTeacherAsync(teacher);
        return CreatedAtAction(nameof(GetTeachers), new { id = created.Id }, created);
    }
}
```
